# Зелья

Зелья - это расходные предметы, дающие персонажу на определенное время специальные эффекты которые можно использовать в игровом процессе

## Способы получения
- Сварить самостоятельно с помощью зельеварки
- Получить с ивентов
- Получить в [Кастомных Данжах](https://funtime.wiki/custom_dungeons)
- Приобрести в /donshop

## Особые зелья в донат-магазине

В /donshop можно приобрести специальные готовые зелья, которые можно также получить с способов которые указывали ранее

# Усиление зелий
Почти любое зелье можно усовершенствовать, повысив его уровень. Для этого нужно поместить два зелья одинакового уровня и типа в наковальню. Игрок потратит уровень опыта и получит взамен одно улучшенное зелье более высокого уровня.

```
Пример : Зелье Силы II + Зелье Силы II = Зелье Силы III
```
