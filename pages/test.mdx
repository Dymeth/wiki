## Списки

### Неупорядоченный список

- Первый элемент
- Второй элемент
  - Вложенный элемент
  - Ещё один вложенный
- Третий элемент

### Альтернативный неупорядоченный список

* Первый элемент
* Второй элемент
  * Вложенный элемент

### Упорядоченный список

1. Первый элемент
2. Второй элемент
   1. Вложенный элемент
   2. Ещё один вложенный
3. Третий элемент

### Смешанный список

1. Первый пункт
   - Подпункт
   - Ещё подпункт
2. Второй пункт
   - Подпункт

### Встроенный код

### Блок кода без синтаксиса

```
Это простой блок кода
```

### Блок кода с подсветкой синтаксиса (JavaScript)

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

### Блок кода с подсветкой синтаксиса (Python)

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```

### Блок кода с подсветкой синтаксиса (HTML)

```html

  </head>
  <body>
    <h1>Привет, мир!</h1>

```
