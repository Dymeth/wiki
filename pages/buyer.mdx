# Скупщик
Специальный торговец который принимает определенные товары за специальную сумму, пользоваться им можно на спавне возле аукциона или же по команде /buyer. Сам скупщик принимает определенные товары и за их количество дает монеты

Список предметов **разделен** на несколько категорий :
1. Еда
2. Фермер
3. Ресурсы
4. Лут с мобов


Также существует специальная категория скупщика которая обновляется раз в 3 часа, и предметы меняются — **Горячие товары**
